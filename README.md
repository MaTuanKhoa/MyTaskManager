This project is a simple single-page **Task Manager application** built using **.NET MAUI** and **XAML**.  
The application demonstrates the use of **data binding** and the **MVVM pattern** with the help of **CommunityToolkit.Mvvm**.

Users can:
- Enter a new task
- Add tasks to a list
- Mark tasks as completed
- View task statistics (total tasks and completed tasks)
